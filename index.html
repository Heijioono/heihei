ようこそ！

このサイトは、GitHubが提供するクラウド開発環境「Codespaces」と、AIコーディングアシスタント「Gemini」を連携させて開発・公開したデモンストレーションページです。

最新の開発ツールがいかにアイデアの具現化と公開を加速させるか、その一例としてご覧ください。

このサイトの目的

目的は、最新の開発ツールを使って、アイデアを素早く形にし、世界中に公開するプロセスを実証することです。VSCodeライクな高機能エディタをブラウザ上で利用し、AIのサポートを受けながらコーディング。編集内容はLive Serverでリアルタイムに確認し、完成したサイトはGitHub Pagesを通じて簡単に公開できます。

開発プロセス

  1. Codespacesの起動: GitHubリポジトリから直接クラウド上の開発環境を起動します。
  2. Gemini拡張機能の追加: VSCodeマーケットプレイスからGemini拡張機能を追加し、AIによるコーディング支援を受けられるようにします。
  3. 開発とプレビュー: HTML, CSS, JavaScriptを編集し、Live Server拡張機能でリアルタイムにブラウザで表示を確認します。
  4. GitHub Pagesでの公開: 変更をGitHubにプッシュし、GitHub Pagesの設定を有効にすることで、サイトをインターネット上に公開します。
